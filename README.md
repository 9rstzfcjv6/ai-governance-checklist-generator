# AI Governance Checklist Generator

AI Governance Checklist Generator is a Streamlit-based prototype designed to generate operational AI governance checklists based on the type of AI system being reviewed.

The project focuses on translating AI governance risks into practical control requirements across areas such as data governance, transparency, human oversight, documentation, monitoring, vendor risk and IP/data use.

## Project Status

Version: `v0.1`

Status: Project setup completed. Core governance logic to be added.

## Planned Features

- AI system type selection
- Governance checklist generation
- Risk level assignment
- Recommended controls
- Responsible owner mapping
- Implementation priority
- Markdown and DOCX report export
